Basic Usage

  1. Add Cycle
  2. Edit Cycle

Add Cycle

  1. Tap add button
  2. Set your period start date in "START DATE" and the period in "LENGTH"
  3. Tap "SAVE" button

The initial value of LENGTH can be changed in the settings.

Edit Cycle

  1. Tap list button
  2. Swipe left on any row and tap the pencil mark button
  3. Tap "SAVE" after modification

If the value is not helpful due to medication, pregnancy, etc., turn on the "Exclude this period from prediction" switch.
