How to change the priority for your data sources

If you use multiple data sources in Apple's Health app, data will get nicely combined in order of priority. You can change the priority order in the Health app.

  1. Open the Health app.
  2. Tap Health Data.
  3. Go to the data type page.
  4. Tap Share Data.
  5. Tap Edit at the navigation bar.
  6. Drag a data source to change order.
