Premiumの購入を復元する

Premiumを購入済みの場合、購入画面にある「購入を復元する」ボタンにて復元ができます。

復元を試みても何も起こらない場合

  • 購入時に使ったのと同じ Apple ID を使ってサインインしていることを確認します。必要であれば、別の Apple ID でサインインします。
  • デバイスの機能制限を確認して、App 内課金が許可されていることを確かめます。
  • デバイスを再起動します。

「Regrant Failed」と表示される場合

Regrant Failed
Please purchase the software application first.

このメッセージは、アプリをダウンロードして購入を復元しようとするときに2つの異なるApple IDが使用された場合に発生します。
以下の手順をお試しください。

  1. App Storeからサインアウトする
  2. 正しいアカウントでサインインしなおす
  3. iPhoneを再起動する
  4. 購入の復元を試す

上記で解決できない場合は、アプリを削除し再インストールしてください。
