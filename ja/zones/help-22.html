PROの購入を復元する

PROをすでに購入済みの場合、新たに購入することなく復元することが可能です。
(PROは非消費型のApp内課金アイテムです)

復元方法

  1. 「PROへアップグレード」の画面を表示します。
    この画面は、タブの「スタッツ」などから表示できます。
  2. 「購入の復元」ボタンをタップします。

復元を試みても何も起こらない場合

  • 購入時に使ったのと同じ Apple ID を使ってサインインしていることを確認します。必要であれば、別の Apple ID でサインインします。
  • デバイスの機能制限を確認して、App 内課金が許可されていることを確かめます。
  • デバイスを再起動します。

「Regrant Failed」と表示される場合

Regrant Failed
Please purchase the software application first.

このエラーメッセージは、アプリをダウンロードして購入を復元しようとするときに2つの異なるApple IDが使用された場合に発生します。

  1. App Storeからサインアウトする
  2. 正しいアカウントでサインインしなおす
  3. iPhoneを再起動する
  4. Zonesアプリで購入の復元を試す
上記で解決できない場合は、アプリを削除し再インストールします。
アプリを削除する際には、必ずヘルスケアデータを「残す」を選択するように注意してください。（詳しくは、アプリの削除/機種変更を確認ください。）
